Together Indy CAN Website

Redesigned the Marion County Public Health Department FIMR Program website to improve accessibility, navigation, and visual design using Figma and WordPress.

Live Site

Objective

The goal was to modernize and restructure the outdated FIMR Program website to improve user navigation, accessibility, and content presentation, ensuring the community can easily access public health resources and information.

Development Process

💡 Planning & Ideation

Conducted stakeholder interviews with program managers, researched user needs, and created wireframes and prototypes in Figma. Developed a new site map and page hierarchy to improve navigation and content discoverability.

💡 Implementation

Built the site in WordPress using custom CSS for styling, implemented interactive features like events and resource pages, and optimized accessibility and SEO for better user engagement. Tracked analytics to guide iterative improvements.

💡 Challenges & Solutions

Technical Challenge: Updating a very basic and outdated WordPress site with limited structure while keeping all content accessible.

The Solution: Restructured the site architecture, created new page templates in WordPress, and used Figma prototypes to ensure smooth user flow and accessibility compliance.

Project Impact

The redesigned website increased usability, engagement, and accessibility, connecting users more effectively with essential public health resources.

  • Implemented interactive resource and events pages to enhance community engagement.
  • Mastered WordPress customization, Figma prototyping, and accessibility-focused design.

Development Reflection

Reflecting back on my development process, the project grew from a basic, outdated WordPress site to a fully redesigned, user-centered platform that improves access to critical public health information while providing a polished and accessible interface.
