Indiana Phenology Wireframes

Developed early-stage designs for an accessible, user-centered K-12 educational resource platform.

Figma Prototype

Objective

To research, plan, and create wireframes and style guides for a K-12-focused educational website, ensuring accessibility, usability, and visual consistency to prepare the project for development in the next semester.

Implementation

Conducted competitive analyses and user research to inform design direction. Created a comprehensive site map and high-fidelity style guide in Figma to ensure consistent branding and accessibility. Developed low-fidelity wireframes and evolved them into interactive high-fidelity mockups, demonstrating layouts, visual components, and functionality for the educational resource site.

Results

Produced a fully documented design framework ready for development. Provided client with visual assets, interactive prototypes, and a clear style guide to guide implementation.

  • High-fidelity wireframes demonstrating accessible layouts and visual hierarchy.
  • Comprehensive style guide ensuring consistent branding and usability across the platform.
  • Figma prototype showcasing interactive user flows for early stakeholder review.
