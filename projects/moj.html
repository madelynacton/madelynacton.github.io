Ministry of Jokes

A full-stack web app for joke submission, rating, AI-assisted moderation, and admin management.

GitHub

Objective

Build a secure full-stack Flask web application that allows users to submit jokes, rate others’ jokes, and provides admin tools for moderation and role management. The app integrates AI-assisted joke rating, activity logging, and robust user authentication. Deployment is containerized using Docker, with database migrations handled via Alembic and SQLAlchemy ORM for PostgreSQL.

Development Process

💡 Planning & Ideation

Mapped out features including joke submission, rating, user profiles, admin actions, and activity logging. Designed the database schema with SQLAlchemy and user flows for registration, login, and ratings.

💡 Implementation

Implemented the backend using Flask with SQLAlchemy ORM and Alembic for migrations. Used Flask-Login for authentication, Flask-WTF for form handling, and Jinja2 templates for dynamic rendering. Routes were modularized for user, admin, and API functionality. Docker was used for containerized deployment with Gunicorn as the WSGI server.

💡 Challenges & Solutions

Challenge: Preventing users from rating their own jokes and ensuring each joke can only be rated once per user.

Solution: Developed a custom decorator to enforce author restrictions and added validation in the rating API. Unit and functional tests confirmed correctness.

Project Impact

Created a dynamic joke-sharing platform with full CRUD functionality, secure authentication, role-based access control, AI-assisted ratings, and admin moderation tools.

  • Full-stack Flask application with secure user authentication and role-based access.
  • Implemented many-to-many relationships in SQLAlchemy for rating system.
  • Gained hands-on experience with Python decorators, Flask-WTF, Jinja2, Docker, and CI/CD deployment.

Development Reflection

The project evolved from a basic joke submission idea to a fully-featured, containerized web app with AI-assisted ratings, admin moderation, secure authentication, and database migrations. It reinforced full-stack development skills, backend design patterns, and team collaboration.
